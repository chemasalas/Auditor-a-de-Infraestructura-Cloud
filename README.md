# Auditor-a-de-Infraestructura-Cloud
Este repositorio es de la actividad de Auditoría de Infraestructura Cloud.
He utilizado XPath para buscar información dentro del archivo CatalogoCloud.xml
En el archivo .xbook se encuentran todas las consultas hechas.

Las consultas son:
- Buscar puertos de servicios en un centro de datos concreto
- Consultar la versión del sistema operativo de un servidor
- Localizar discos de gran capacidad
- Identificar el primer servidor apagado
- Obtener la arquitectura de CPU de un servidor con GPU

El trabajo lo he hecho usando VS Code y la extensión XPath Notebook
